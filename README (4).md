# Employee Management System

## Description

This is a simple Python project based on Object-Oriented Programming
(OOP).\
It allows users to create and manage Employees, Managers, and
Developers.

------------------------------------------------------------------------

## Features

-   Create Employee\
-   Create Manager (with department)\
-   Create Developer (with programming language)\
-   Display latest details\
-   Menu-driven program using loop

------------------------------------------------------------------------

## Classes Used

### Employee

-   Attributes: name, age, employee_id, salary\
-   Methods:
    -   display()\
    -   get_employee_id()

### Manager (inherits Employee)

-   Additional attribute: department\
-   Overrides display()

### Developer (inherits Employee)

-   Additional attribute: programming language\
-   Overrides display()

------------------------------------------------------------------------

## How the Program Works

1.  User selects an option from the menu\
2.  Enters required details\
3.  Object is created and stored in a list\
4.  User can view details\
5.  Program runs until exit
